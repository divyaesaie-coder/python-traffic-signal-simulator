# python-traffic-signal-simulator
A simple Python traffic signal simulator using loops and time delays to switch between Red, Green, and Yellow signals automatically.
# Python Traffic Signal Simulator

A beginner-friendly Python project that simulates a traffic signal system using loops and timers.

## Features
- Displays RED, GREEN, and YELLOW signals
- Automatic signal switching using time delays
- Infinite loop simulation
- Simple and easy-to-understand code

## Technologies Used
- Python 3
- time module

## How It Works
The program continuously changes traffic signals in this order:
1. RED
2. GREEN
3. YELLOW

Each signal stays active for 5 seconds before switching.

## Concepts Used
- while loop
- time.sleep()
- Sequential execution
- Console output

```bash
